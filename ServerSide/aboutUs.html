About Us

Company Info

Educaional Empowerment was founded in Education, Ohio by Sean Adkins and Isaac Bennett. E.E. is a startup company built around the idea of helping students improve their test scores by providing free, automatic tutoring.

Built on the concept that some students need help. Educaional Empowerment utilizes server tech to handle students and allow them to improve on their own time.

Company Members

Sean Adkins: Co-Head Web Developer, Front End Developer

Isaac Bennett: Co-Head Web Developer, Back End Developer

Goal for this website

This website was built with the intention of automating the process of assisting students in their acadeic careers.
